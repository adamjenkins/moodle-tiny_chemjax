# ChemJax TinyMCE Plugin - Changes

## 0.1.0 (2026-07-16)

Initial TinyMCE rewrite of the legacy Atto ChemJax plugin. This version scaffolds the plugin structure with:

- TinyMCE plugin interface implementation
- ChemJax formula dialog with live preview
- Integration with the ChemJax filter for rendering
- Multi-language support (English, Japanese)
- Capability-based access control
- Privacy provider implementation
