# Changelog

All notable changes to the ChemJax TinyMCE plugin will be documented in this file.

## [0.1.0] - 2026-07-16

### Added

- Initial TinyMCE implementation of the ChemJax editor plugin
- Dialog interface for entering ChemJax chemistry notation
- Live preview of rendered formulas using the ChemJax filter
- Multi-language support (English and Japanese)
- Capability-based permission system (`tiny/chemjax:use`)
- Privacy provider for GDPR compliance
- Example image resources for documentation
